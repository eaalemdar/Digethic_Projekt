
# Mietpreisanalyse und Vorhersage für Hessen

Dieses Projekt analysiert Immobilien-Daten, um Mietpreise vorherzusagen. Es wurde im Rahmen des Moduls "Digital Ethics" erstellt.

## Inhalt
* `projekt.ipynb`: Das Jupyter Notebook mit dem gesamten Python-Code, der Datenbereinigung und den Modellen.
* `/Data`: (Lokal) Enthält den Datensatz `immo_data.csv`.

## Wichtiger Hinweis zu den Daten
Aufgrund der Dateigrößenbeschränkung von GitHub (100 MB) ist die Datendatei `immo_data.csv` (ca. 272 MB) **nicht** in diesem Repository enthalten. 

**Um den Code auszuführen:**
1. Laden Sie den Datensatz `immo_data.csv` herunter.
2. Erstellen Sie lokal einen Ordner namens `Data`.
3. Legen Sie die Datei in diesen Ordner, damit das Notebook darauf zugreifen kann.
